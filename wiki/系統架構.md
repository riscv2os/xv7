# 系統架構

xv7 是基於 xv6-riscv 的 Unix 風格作業系統，額外新增了 TCP/IP 網路功能。

## 系統架構

```
┌─────────────────────────────────────────────────────────┐
│                    使用者空間                            │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐        │
│  │ tcpecho │ │ httpd   │ │ telnetd │ │ curl    │        │
│  └─────────┘ └─────────┘ └─────────┘ └─────────┘        │
├─────────────────────────────────────────────────────────┤
│                    系統呼叫                              │
│  open, read, write, fork, exec, pipe, socket, connect   │
├─────────────────────────────────────────────────────────┤
│                    核心空間                              │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐        │
│  │ 行程管理    │ │ 檔案系統    │ │ 網路堆疊    │        │
│  │             │ │ (SFS)       │ │             │        │
│  └─────────────┘ └─────────────┘ └─────────────┘        │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐        │
│  │ 記憶體管理  │ │ 裝置驅動    │ │ VirtIO      │        │
│  │             │ │            │ │ 網路/磁碟   │        │
│  └─────────────┘ └─────────────┘ └─────────────┘        │
└─────────────────────────────────────────────────────────┘
```

## 開機流程

1. **開機 ROM** (0x1000)：QEMU 載入核心並跳躍至核心進入點
2. **進入點** (0x80000000)：kernel/entry.S 初始化堆疊
3. **主函數** (kernel/main.c)：
   - `kinit()` - 實體頁面配置器
   - `kvminit()` - 建立核心頁表
   - `procinit()` - 行程表初始化
   - `trapinit()` - 陷阱向量設定
   - `binit()` - 緩衝區快取
   - `iinit()` - inode 表
   - `netinit()` - 網路堆疊（xv7 新增）
   - `userinit()` - 第一個使用者行程

4. **排程器**：輪詢式行程排程開始

## 記憶體配置

| 位址 | 說明 |
|------|------|
| 0x1000 | 開機 ROM |
| 0x02000000 | CLINT（計時器） |
| 0x0C000000 | PLIC（中斷控制器） |
| 0x10000000 | UART0（序列埠） |
| 0x10001000 | VirtIO 磁碟/網路 |
| 0x80000000 | 核心程式碼 |

## 關鍵元件

- **行程管理**：`kernel/proc.c`、`kernel/proc.h`
- **記憶體管理**：`kernel/vm.c`、`kernel/kalloc.c`
- **檔案系統**：`kernel/fs.c`、`kernel/bio.c`
- **網路堆疊**：`kernel/net/`
- **裝置驅動**：`kernel/virtio_disk.c`、`kernel/uart.c`

## 執行 xv7

```shell
cd /Users/Shared/ccc/c0py/os/xv7
make qemu
```

在 xv7 內：
```shell
ifconfig net0 192.0.2.2 netmask 255.255.255.0
tcpecho
```
