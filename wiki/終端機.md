# 終端機

xv6/xv7 的終端機系統負責處理序列埠的輸入輸出，提供互動式的命令列介面。

## 硬體

- **UART 晶片**：QEMU 模擬的 16550 相容 UART
- **MMIO 位址**：0x10000000 (UART0)
- ** IRQ**：10

## 終端機緩衝區

```c
struct {
  struct spinlock lock;
  
  // 輸入緩衝區
  #define INPUT_BUF_SIZE 128
  char buf[INPUT_BUF_SIZE];
  uint r;  // 讀取索引
  uint w;  // 寫入索引
  uint e;  // 編輯索引
  
  // 輸出
  int consolefd;  // 檔案描述符
} cons;
```

## 輸入處理

終端機支援特殊控制字元：

| 字元 | 說明 |
|------|------|
| `Ctrl+H` (0x08) | 退格鍵 |
| `Ctrl+U` (0x15) | 清除整行 |
| `Ctrl+D` (0x04) | 檔案結束 (EOF) |
| `Ctrl+P` (0x10) | 顯示行程列表 |
| `Enter` | 執行命令 |
| `Ctrl+C` (0x03) | 終止目前行程 |

## 輸出函數

```c
void consputc(int c);     // 輸出單一字元
void consoleintr(int c);  // 處理輸入中斷
```

## 序列輸出

UART 驅動程式 (`kernel/uart.c`)：

```c
void uartinit(void);           // 初始化 UART
void uartputc_sync(int c);    // 同步輸出字元
int uartgetc(void);           // 讀取字元
void uartintr(void);          // 中斷處理
```

## 檔案介面

終端機作為特殊檔案處理：

- 標準輸入 (stdin)：檔案描述符 0
- 標準輸出 (stdout)：檔案描述符 1
- 標準錯誤 (stderr)：檔案描述符 2

## 相關檔案

- `kernel/console.c` - 終端機邏輯
- `kernel/uart.c` - UART 驅動程式
- `kernel/uart.h` - UART 定義
