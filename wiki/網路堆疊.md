# 網路堆疊

xv7 新增的 TCP/IP 網路堆疊是基於 [microps](https://github.com/pandax381/microps) 專案，實作於核心空間。

## 網路堆疊架構

```
+------------------+
|   Socket API     |  (syssocket.c)
+------------------+
|   TCP / UDP      |  (tcp.c, udp.c)
+------------------+
|   IP / ICMP / ARP|  (ip.c, icmp.c, arp.c)
+------------------+
|   Ethernet       |  (ether.c)
+------------------+
|   VirtIO Net    |  (virtio_net.c)
+------------------+
```

## 網路裝置

```c
struct net_device {
    char name[IFNAMSIZ];     // 裝置名稱 (如 "net0")
    uint16_t type;          // 裝置類型 (ETHERNET)
    uint16_t mtu;           // 最大傳輸單元 (1500)
    uint16_t flags;         // 狀態旗標 (UP, RUNNING, etc.)
    uint8_t addr[16];       // MAC 位址
    uint8_t broadcast[16];  // 廣播位址
    void *priv;             // 私有資料
};
```

## 通訊協定

### Ethernet
- 乙太網路 frame 處理
- MAC 位址處理
- 協定類型（IPv4: 0x0800, ARP: 0x0806）

### ARP
- IP 到 MAC 位址解析
- 維護 ARP 表格

### IP
- IP 封包處理
- 分片與重組
- 路由（簡單靜態路由）

### ICMP
- Ping 請求/回應
- 錯誤訊息

### TCP
- 連接導向傳輸
- 流量控制
- 壅塞控制
- 狀態機：LISTEN, ESTABLISHED, TIME_WAIT 等

### UDP
- 無連接傳輸
- 簡單的封裝/解封裝

## 網路初始化流程

```c
// kernel/main.c
netinit();        // 初始化網路堆疊
virtio_net_init(); // 初始化網路卡驅動
netrun();        // 啟動網路事件處理
```

## Socket API

xv7 實作 BSD 風格的 Socket 介面：

```c
socket(int domain, int type, int protocol);
// domain: AF_INET (IPv4)
// type: SOCK_STREAM (TCP), SOCK_DGRAM (UDP)
// protocol: 0 (自動選擇)

bind(int sockfd, struct sockaddr *addr, socklen_t addrlen);
listen(int sockfd, int backlog);
accept(int sockfd, struct sockaddr *addr, socklen_t *addrlen);
connect(int sockfd, struct sockaddr *addr, socklen_t addrlen);
send(int sockfd, const void *buf, size_t len, int flags);
recv(int sockfd, void *buf, size_t len, int flags);
close(int sockfd);
```

## 使用方式

在 xv7 內設定網路：
```shell
ifconfig net0 192.0.2.2 netmask 255.255.255.0
```

測試範例：
```shell
tcpecho     # TCP Echo 伺服器 (port 7)
udpecho     # UDP Echo 伺服器
httpd       # HTTP 伺服器
telnetd    # Telnet 伺服器
curl        # HTTP 用戶端
```
